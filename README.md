# zim-music-vue3

Vue 3 + TypeScript + Element Plus + Vite 开发的音乐网站 🎧

后端使用: [网易云音乐 Node.js API](https://github.com/Binaryify/NeteaseCloudMusicApi)

- [02.搭建整体架构](./doc/02.搭建整体架构.md)

## 📚 使用方法

clone:

```bash
git clone git@github.com:ZimTang/zim-music-vue3.git
```

install:

```bash
pnpm install
```

run:

```bash
pnpm run dev
```

build

```bash
pnpm run build
```
